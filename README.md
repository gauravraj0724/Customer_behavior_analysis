# 📊 Customer Behavior Analysis & Dashboard Project

## 🔍 Overview

This project focuses on analyzing customer behavior and retail sales data to uncover meaningful insights and support business decision-making.
The project includes **data cleaning, exploratory data analysis (EDA), SQL querying, and Power BI dashboard visualization**.

---

## 🎯 Objectives

* Analyze customer purchasing patterns
* Identify sales trends and seasonality
* Understand customer behavior
* Generate insights for business growth

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* Jupyter Notebook
* SQL (pgAdmin)
* Power BI
